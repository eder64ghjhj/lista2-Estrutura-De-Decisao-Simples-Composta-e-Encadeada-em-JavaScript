# 🔀 Lista 2 — Estrutura de Decisão Simples, Composta e Encadeada em JavaScript

Série de 20 exercícios práticos focados em estruturas condicionais em JavaScript, explorando o controle de fluxo através de condicionais simples, compostas e encadeadas.

## 📚 Conteúdo

- Condicionais Simples (`if`)
- Condicionais Compostas (`if...else`)
- Condicionais Encadeadas (`if...else if...else`)
- Operadores lógicos (`&&`, `||`, `!`) e de comparação (`==`, `===`, `>`, `<`, etc.)
- Resolução de problemas com tomada de decisão dinâmica

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
